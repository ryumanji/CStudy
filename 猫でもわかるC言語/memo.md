◆エスケープシーケンス
\n : 改行
     現在の印字位置を次の行の先頭位置に移動する
\t : タブ
     次の水平位置タブに移動する
\b : バックスペース
     現在の行で前に移動する。先頭にある場合は不定
\r : キャリッジリターン
     現在行の最初の位置に移動
\f : ページフィールド
     次の論理ページの最初に移動
\' : シングルクォーテーション
     一重引用符
\" : ダブルクォーテーション
     二重引用符
\0 : ヌル文字
\\ : 円記号
\? : クエスチョンマーク
\a : ベル音
\xhh : 16進拡張
       16進でhhのコードを持つ文字
\ooo : 8進拡張
       8進でoooのコードを持つ文字

◆scanf関数の変換指定子
%c : 文字入力
%d : 10進整数
%f : float型浮動小数点数
%lf : double型浮動小数点数
%s : 文字列
%p : ポインタ
%o : 8進数
%x : 16進数
%u : 符号なし10進数

◆fopen関数のオープンモード
r : 読み出し用にオープン
　　指定したファイルが存在しないとき、関数は失敗する
w : 書き込み用にオープン
　　指定したファイルが既に存在するときは、これに上書きされる
　　指定したファイルが存在しないときは、新規にファイルが作成される
a : すでに存在するファイルに追加する
　　指定したファイルが存在しないとき、新規にファイルが作成される
r+ : ファイル読み込み・更新モード
　　　指定したファイルが存在しないとき、関数は失敗する
w+ : 読み出し書き込みモードでオープン
　　　指定したファイルが既に存在するとき、これに上書きされる
a+ : 読み出し、追加・更新モード
rb : バイナリで読み込み
wb : バイナリで書き込み
ab : バイナリで追加
